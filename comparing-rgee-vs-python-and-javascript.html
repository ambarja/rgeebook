Comparing rgee vs Python and Javascript

This section requires the next libraries:

library(rgee)

ee_Initialize()

The R, Python and JavaScript APIs access the same server-side functionality, but client-side expressions (learn more about client vs. server) can vary because of language syntax differences. The Earth Engine Python API and rgee share the same modules, classes, functions, and methods. In other words, the logic of the syntax (just change . by a $) is the same and the execution time is just as fast. However, keep in mind that syntactic differences may exist in certain situations. The following table includes a list of the common syntax differences you’ll encounter when working with the R API relative to the Python and JavaScript API.

Function definition

  • JavaScript
function myFun(arg) {
  return arg;
}

var myFun = function(arg) {
  return arg;
};
  • Python
def my_fun(arg):
  return arg
  • R
my_fun <- function(arg){
  return(arg)
}

Anonymous function mapping

  • JavaScript
var foo = col.map(function(arg) {
  return arg;
});
  • Python
foo = col.map(lambda arg: arg)
  • R
foor <- lapply(col, function(arg) arg)

Variable definition

  • JavaScript
var myVar = 'var';
  • Python
my_var = 'var'
  • R
my_var <- 'var'

Logical operators

  • JavaScript
var match = such.and(that);
var match = such.or(that);
var match = such.not(that);
  • Python
match = such.And(that)
match = such.Or(that)
match = such.Not(that)
  • R
match <- such$And(that)
match <- such$Or(that)
match <- such$Not(that)

Multi-line method chain

  • JavaScript
var foo = my.really()
              .reallyLong()
              .methodChain();
  • Python
foo = (my.really()
       .reallyLong()
       .methodChain())
  • R
foo = my %>% 
  really() %>% 
  reallyLong() %>% 
  methodChain()

Dictionary keys

  • JavaScript
var dic = {'key': value};
var dic = {key: value};
  • Python
dic = {'key': value}
  • R
dic <- list(key = value)

Dictionary object access

  • JavaScript
var value = dic.key;
var value = dic['key'];
  • Python
value = dic['key']
  • R
value <- dic[['key']]

Function argument definition

  • JavaScript
// Positional arguments.
var foo = fun(argX, argY, argZ);
// Keyword arguments object.
var foo = fun({y: argY});
  • Python
# Positional arguments.
foo = fun(arg_x, arg_y, arg_z)
# Keyword arguments dictionary.
foo = fun(**{'y': arg_y})
# Keyword arguments.
foo = fun(x=arg_x, z=arg_z)
  • R
# Positional arguments.
foo <- fun(arg_x, arg_y, arg_z)

# Keyword arguments.
foo <- fun(x=arg_x, z=arg_z)

Boolean

  • JavaScript
    var t = true;
var f = false;
  • Python
t = True
f = False
  • R
t <- TRUE
f <- FALSE

Boolean

  • JavaScript
var na = null;
  • Python
na = None
  • R
na <- NULL

Comment

  • JavaScript
//
  • Python
#
  • R
#
